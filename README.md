# Python-car-wash
Finals project for programming in Python

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application:
   ```bash
   python main.py
   ```

## Development Tools
- **Populate Fake Data**: To add fake customers, cars, and staff for testing purposes, run:
  ```bash
  python populate_fake_users.py
  ```
